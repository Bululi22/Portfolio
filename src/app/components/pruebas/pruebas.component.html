FJ

Aplicación web interactiva desarrollada con Angular 17 para mejorar la velocidad y precisión al escribir. Presenta una interfaz minimalista, intuitiva y enfocada en la enseñanza eficaz de la mecanografía.


Incluye:

  • Línea de texto con retroalimentación en tiempo real para mejorar la escritura.
  • Teclado virtual que resalta la tecla que debe pulsarse para guiar al usuario.
  • Guía visual de manos que indica qué dedo utilizar para cada tecla, facilitando el aprendizaje.

Diseñada para aprender mecanografía de forma intuitiva y efectiva.

Informula

Aplicación web desarrollada con Angular 19 que muestra información actualizada sobre la temporada de Fórmula 1. Presenta una interfaz clara y funcional con los datos más relevantes, obtenidos desde una API externa (no oficial).


Incluye:

  • Contador regresivo en tiempo real para la próxima carrera.
  • Clasificación actual de pilotos y equipos.
  • Diseño moderno con Tailwind CSS y DaisyUI.

Gesclub

Aplicación web desarrollada con Angular 19 que facilita la gestión de socios en un club deportivo. Presenta una interfaz intuitiva y eficiente que permite añadir, eliminar y modificar.


Incluye:

  • Gestión completa de socios con perfiles personalizados.
  • Calendario de eventos y actividades con notificaciones automáticas.
  • Diseño moderno con Tailwind CSS y DaisyUI.
